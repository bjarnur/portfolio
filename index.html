King of the Cube

The King of the Cube is a multiplayer, augmented reality game inspired by classic games such as Donkey Kong. The project started as a group assignment in the course Advanced Graphics and Interaction at KTH. Together with five other students, I worked on the game between August and December 2018. Some of us are still working on the project today, and we have plans to make it available to the public this year.

My main responsibilities on the project have been working on networking features, so that the game can support multiple players, and on custom graphics, such as smoke and explosions. I have also been involved in implementing other game features, such as character movement and game rules. The project is implemented in the Unity game engine, and uses Google's ARCore software development kit to support the AR functionality.

A Tonne of Trees

A project in the course Advanced Topics in Computer Graphics, the aim of the project was to reproduce some of the methods presented in the paper Real-time Realistic Rendering and Lighting of Forests by Bruneton and Neyret. The system I created can render impostors based on a 3D model. For each of the sample images the system registers depth from the mesh surface to the model center, this depth information can be used to render complex lighting effects on the impostors. However, implementing the lighting effects was outside the project scope.

The project is written in C++ with modern OpenGL. It served as a great opportunity for me to get familiar with modern OpenGL, and throughout the project work I got a lot of help from learnopengl.com.
